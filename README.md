# Pneumonia Detection using CNN

## Objective
Classify chest X-ray images into Pneumonia or Normal.

## Dataset
- 5863 X-ray images
- Categories: Pneumonia / Normal

## Approach
- Image preprocessing (rescaling)
- CNN model (Conv + Pool + Dense layers)
- Binary classification

## Model Architecture
- Conv2D
- MaxPooling
- Dense layers
